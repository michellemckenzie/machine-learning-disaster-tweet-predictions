Predicting if Tweets are about Real Disasters

Info

We have utilized Natural Language Processing (NLP) to scan thousands of tweets to detect if a tweet is about a real disaster. After we cleaned the data, we then trained and tested multiple models until we settled on the SVM model. SVM has the best precision for deciding a disaster with a precision average of about 82%. To interact with our model, please type and enter a sentence down below.

Dataset Description

The dataset that we chose comes from Kaggle. Examples of real disasters include: forest fires, tornados, hurricanes, tsunamis, earthquakes, shootings, bombings, etc. The dataset contains five columns: id, keyword, location, text, target. We only utilized the text and target columns. The text is the raw tweet, and the target indicates if it is real (1) or not (0).
